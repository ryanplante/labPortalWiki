About

This is a portal for all students and facility to use to maintain and get information about labs for New England Tech.

This was created so that students who are hired as tutors and lab monitors can log information about who uses the lab and what items they borrow for book keeping purposes.

This project is an addition to Lab Monitoring tool, with the aim for adding more features, and improving the code and security from what we learned throughout our endeavour of our Bachelor's Degree

Features

  • Login system with enhanced security
  • Scheduling
  • User privilege system
  • Tutor chat
  • Mobile support
  • Barcode scanning

  • Ability to edit (add to, delete from, modify) the tables keeping track of multiple aspects of the lab monitoring system
